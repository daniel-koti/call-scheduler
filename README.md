# Call Scheduler

Application that connects your Google calendar and allows people to book appointments in their free time.

## Tools
- Next.js
- Typescript
- Tailwindcss
- Prisma
- Axios
- Next Auth
- Zod
- React Hook Form

## Important learnings
- Social authentication (Google)

## Features
- Create user
- Connect google account
- Google calendar permissions validation
